## 🚀 Descripción del Proyecto
Este repositorio contiene la arquitectura rediseñada del sistema **Colmena**, transformando un estudio de caso UX estático en una aplicación web interactiva y escalable. 

El proyecto aplica una **metodología de ingeniería de diseño industrializada**, utilizando una "Single Source of Truth" (design.md) para garantizar la paridad total entre el diseño y el código final.

## 🧠 Valor Diferencial
- **Diseño basado en Tokens:** Implementación de reglas gobernadas por un archivo maestro, definiendo 36 tokens (Primitivos, Semánticos y de Componentes).
- **Componentes Reutilizables:** Arquitectura de Angular que elimina código duplicado y facilita el mantenimiento.
- **Accesibilidad (WCAG AA):** Contraste y jerarquía visual optimizados para usuarias con baja alfabetización digital.
- **IA-Driven Development:** Desarrollo orquestado mediante **Cursor Agent** y **Model Context Protocol (MCP)**.

## 🛠️ Stack Tecnológico
- **Frontend:** Angular (Standalone Components)
- **Estilos:** Tailwind CSS v3 & DaisyUI
- **Estandarización:** Design Tokens (YAML)
- **Control de Versiones:** Git con estándares *Conventional Commits*
